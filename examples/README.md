# A2AL Examples

Worked examples demonstrating each archetype, canonical encoding, and Markdown→A2AL transpilation.

> **Status:** Planned. See the issue tracker for current authoring priorities.

## Planned Contents

- `archetypes/` — minimal and realistic example messages for each of the seven archetypes (A1–A7), with canonical bytes and human-readable annotations
- `transpilation/` — paired Markdown documents and their A2AL equivalents, demonstrating token reduction
- `roundtrip/` — examples showing a writer → relay → reader chain that preserves unknown codes per `specs/A2A-Grammar.md` §8

Examples are normative at the level of "here is canonical output for this input" but do not extend the grammar.
